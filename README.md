# 텍스트 방향 변환기

Excel 열(column) 데이터를 한 줄로, 또는 한 줄 텍스트를 여러 줄로 변환하는 단일 페이지 웹 앱입니다.

## 실행 방법

`index.html` 파일을 브라우저에서 바로 열면 됩니다. 외부 의존성 없이 오프라인에서도 동작합니다.

## 주요 기능

### 세로 → 가로 (Excel 열 붙여넣기)

여러 줄로 된 데이터를 구분자로 이어 한 줄로 변환합니다.

**입력 예시:**
```
dean
add
kkk
lll
```

**출력 예시 (쉼표 + 접미사 `@linecorp.com`):**
```
dean@linecorp.com, add@linecorp.com, kkk@linecorp.com, lll@linecorp.com
```

### 가로 → 세로

한 줄 텍스트를 분리 기준에 따라 여러 줄로 변환합니다.

**분리 방식:**
- 구분자로 분리 (쉼표, 세미콜론, 파이프, 탭)
- 공백/탭으로 분리
- 고정 글자수로 자르기
- 직접 입력 구분자

## 옵션

| 옵션 | 기본값 | 설명 |
|---|---|---|
| 구분자 | 쉼표 `,` | 쉼표 / 세미콜론 / 파이프 / 공백 / 탭 / 직접입력 |
| 접미사 추가 | (없음) | 각 항목 뒤에 문자열 추가 (예: `@linecorp.com`) |
| 구분자 뒤 공백 | ON | `dean, add` vs `dean,add` |
| 빈 줄 건너뜀 | ON | 빈 줄 무시 |
| 앞뒤 공백 제거 | ON | 각 항목 trim |
| 중복 제거 | ON | 동일한 값은 첫 번째만 유지 |

## 기술 스택

- 순수 HTML / CSS / JavaScript
- 외부 라이브러리 없음
- 다크모드 자동 지원
- 모바일 반응형
